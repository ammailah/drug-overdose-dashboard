# Drug Overdose Deaths Dashboard
**Course:** Exploratory Data Analysis (EDA 350)
**Instructor:** Ali Hassan Sherazi
**Dataset:** VSRR_Provisional_Drug_Overdose_Death_Counts.csv

## How to Run
```
pip install -r requirements.txt
streamlit run app.py
```
Then open http://localhost:8501 in your browser.

## Key Insights
- Overdose deaths increased sharply from 2015 to 2022
- Deaths spiked during COVID-19 (2020-2021)
- California, Florida, and Ohio have the highest death counts
- Heroin deaths declined after 2016 while synthetic opioids surged
